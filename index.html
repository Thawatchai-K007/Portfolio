Smart Bar Stock Management System

Project Description:

I independently developed this real-time inventory system to streamline bar operations. To ensure maximum efficiency, I created separate dedicated modules for different beverage categories.

Public Module: The link below features the live system for Beer, Soft Drinks, Water, and Soda.

Internal Module: I have also developed a comprehensive system for Spirits, Wine, and Liqueurs. However, due to company confidentiality and NDAs (Non-Disclosure Agreements) at my current workplace, this specific module and its data cannot be shared publicly.

Key Features:

  • Live Cloud Sync: Built with JavaScript and Firebase for instant updates during shift transitions.
  • Automated Logic: Includes reorder suggestions and a one-click shift rollover system to reduce manual errors.
  • Security & Privacy: Designed with professional standards to protect sensitive company inventory data.
Live Site (Beer & Soft Drinks): https://barstockbygame.netlify.app/
